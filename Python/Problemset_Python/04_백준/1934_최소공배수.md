# 최소공배수
백준 브론즈1 1934
https://www.acmicpc.net/problem/1934

---
* 유클리드 호제법 : b를 a로 나눈 나머지를 반복적으로 이용하여 최대공약수를 구하는 방법.
* a * b = (a, b의 최소공약수) * (a, b의 최소공배수)

```
T = int(input())
for i in range(T):
    
    # 숫자 입력 받기
    n1, n2 = map(int, input().split())
    
    # 최대공약수 구하기
    a, b = n1, n2
    while b%a != 0:
        a, b = b%a, a
    
    # a * b = (a, b의 최소공약수) * (a, b의 최소공배수)
    answer = int(n1*n2 / a) 
    print(answer)
```
