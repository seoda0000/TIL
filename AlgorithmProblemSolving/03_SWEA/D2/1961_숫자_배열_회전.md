## 1961. 숫자 배열 회전
https://swexpertacademy.com/main/code/problem/problemDetail.do

N x N 행렬이 주어질 때,

시계 방향으로 90도, 180도, 270도 회전한 모양을 출력하라.

```python
T = int(input())

for test_case in range(1, T + 1):
    print(f"#{test_case}")
    N = int(input())
    lst = []    # 행별로 리스트 만들기
    for n in range(N):
        lst.append(list(input().split()))
    
     
    # 가장 위쪽 행이 어떻게 바뀌는지를 기준으로 생각하면 쉽다.
    
    
    # 90도
    # 열 방향. 첫째 열부터 마지막 행에서 시작해서 위쪽 방향으로 읽어야 한다.
    ans1 = []
    for j in range(N):
        l = ""
        for i in range(N):
            l = lst[i][j] + l
        ans1.append(l)

    # 180도
    # 행 방향. 마지막 행부터 마지막 열에서 시작해서 왼쪽 방향으로 읽어야 한다.
    ans2 = []   
    for j in range(N-1, -1, -1):
        l = ""
        for i in range(N):
            l = lst[j][i]+ l
        ans2.append(l)

    # 270도
    # 열 방향. 마지막 열부터 첫번째 행에서 시작해서 아래 방향으로 읽어야 한다.
    ans3 = []
    for j in range(N-1, -1, -1):
        l = ""
        for i in range(N):
            l += lst[i][j]
        ans3.append(l)

    # 각 리스트에서 차례대로 요소 출력
    for i in range(N):
        print(ans1[i], ans2[i], ans3[i])
```

* 90도로 세번 돌리는 풀이법.

```python
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    arr = [list(map(int, input().split())) for _ in range(N)]
    
    # arr 90도 돌려서 arr1 만들기
    arr1 = [[] for _ in range(N)]
    for i in range(N-1, -1, -1):
        for j in range(N):
            arr1[j] += [arr[i][j]]

    # arr1 90도 돌려서 arr2 만들기
    arr2 = [[] for _ in range(N)]
    for i in range(N - 1, -1, -1):
        for j in range(N):
            arr2[j].append(arr1[i][j])
    
    # arr2 90도 돌려서 arr3 만들기
    arr3 = [[] for _ in range(N)]
    for i in range(N - 1, -1, -1):
        for j in range(N):
            arr3[j].append(arr2[i][j])

    print(f'#{tc}')
    for i in range(N):
        print(''.join(map(str,arr1[i])), end = ' ')
        print(''.join(map(str,arr2[i])), end=' ')
        print(''.join(map(str,arr3[i])), end=' ')
        print()
```
